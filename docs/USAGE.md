# Usage (KiCad Global Library)

This repository is intended to be registered as a KiCad **Global** library.

## Symbols
- KiCad: Preferences  Manage Symbol Libraries  Global Libraries
- Add the symbol library file(s) under `symbols/` (when available)

## Footprints
- KiCad: Preferences  Manage Footprint Libraries  Global Libraries
- Add the `.pretty` library folder(s) under `footprints/` (when available)

## 3D Models (optional)
- Keep `3dmodels/` in a stable path and configure KiCad 3D search paths as needed.
